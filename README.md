# Product Filter Table (Frontend Assignment)

Hi, I am **Vinay Ramteke**.  
This project is a **Product Filter Table Dashboard** built using **React (Vite)**. The application fetches product data from DummyJSON API, stores it locally, and simulates API calls using **Mock APIs (Promise + setTimeout)**.
